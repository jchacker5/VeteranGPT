# Veteran Counselor Chatbot

## Description

This project is a chatbot that simulates a Veterans Counselor, using OpenAI and Flask. The chatbot can respond to user inputs in a conversational manner, following a set of rules and requirements provided in the interaction template.

## Installation and Setup

### Prerequisites

- Python 3.10
- Flask
- playsound
- Python-dotenv

### Instructions

1. Clone the repository
2. Navigate to the project directory
3. Install the required dependencies
4. Run the application

## Usage

Once you've started the Flask server, navigate to `http://localhost:5000` in your web browser. Enter your input in the text field and press 'Send' to see the chatbot's response.
